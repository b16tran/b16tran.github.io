Breakdown



Butterfly Effect
I began by animating the butterfly, car crash, and cracks and timed the effects to those events. Main cracks were created using key framed splines and VDB. Smaller cracks were procedurally created. A procedural brick system was created for modeling the building. The destruction was simulated in DOPs using bullet solver with swapped proxy geometry. Constraint types used: pin (sign), cone twist (bench), glue & spring (building). Smaller pieces were simulated using POPs.

Created using Houdini. Responsible for all aspects.


Pigeon Chase
I worked closely with an animator to layout the camera and actions of the characters that would drive the pigeons. The crowd system was simulated using POPs and fuzzy logic was used to determine behavior states. I rigged and animated the pigeon in Houdini. CHOPs was then used to apply the various cycles to the pigeons and blend them together. Tossed seeds were also simulated using POPs.

Created using Houdini. Responsible for pigeon crowd. Kids' animations by Zeina Masri. Kayla/Kyle kid rig by Josh Sobel.


Bubble Bath
Main task of this shot was the procedural suds shader I developed. Faucet water was simulated using POPs. Steam was simulated using smoke solver in DOPs. Everything was rendered using Mantra and composited in Nuke.

Created using Houdini. Responsible for everything except rubber duck model made by Xian Guan.


Roots Bridge
I created a spline-based tool that generates a bridge modeled after the living root bridges found in India. Various parameters including densities for the roots and vegetation, heights for hanging roots, and number of trees can be controlled.

Created using Houdini. Responsible for all aspects.


Fahrenheit 451
I created the book piles using rigid bodies and the book stacks with a procedural system. I created a procedural shader for the books. The fire and smoke were created using the pyro solver. The ash was created using particles.

Created using Houdini. Responsible for the books, fire, smoke, and ashes.
